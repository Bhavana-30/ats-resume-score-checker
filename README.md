# ATS Resume Score Checker

A web application that analyzes a resume against a job description
and calculates an ATS compatibility score.

## Features
- Resume upload (PDF)
- Job description input
- ATS score calculation
- Match type and improvement suggestions

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Google Generative AI
- PyPDF2

## How to Run
1. Clone the repository
2. Create a `.env` file with Google API key
3. Install dependencies using `pip install -r requirements.txt`
4. Run `python app.py`
